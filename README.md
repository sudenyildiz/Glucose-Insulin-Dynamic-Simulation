# Glucose-Insulin Response Simulation
This project demonstrates the mathematical modeling of the human body's regulatory response to glucose intake using MATLAB/Simulink.

## Overview
- **Objective:** To simulate how the pancreas responds to a sudden increase in blood glucose levels.
- **Model:** Utilized Transfer Functions to represent the pharmacokinetic behavior of the insulin-glucose feedback loop.
- **Tools:** MATLAB, Simulink.

## Results
The simulation shows a stable system where the glucose level is regulated back to a steady state, mimicking a healthy biological response.
